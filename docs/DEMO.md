# Demo Assets

Use this folder to store publishable visuals for README:

- `assets/ui-home.png`
- `assets/coach-mode.gif`

Suggested workflow:

1. Start app with `uvicorn cos.app:app --reload`.
2. Open `http://127.0.0.1:8000/`.
3. Capture:
   - home screen screenshot
   - 10-20 second GIF showing ingest -> query -> coach flow
4. Reference files from `README.md`.
