# C-OS v0.1.1

Release date: 2026-05-05

## Summary

v0.1.1 adds LLM-enhanced coaching and simplified next-step recommendations for better user experience.

## New Features

- **Next-Step Recommendations**: Single actionable next step instead of advice lists
- **DeepSeek LLM Integration**: Optional LLM rewriting for more natural coaching language
- **Improved UI**: Streamlined interface with draft saving and suggestion buttons
- **Enhanced API**: New `/coach/next-step` endpoint and `/diagnostics/llm` status

## Configuration

To enable LLM features, set environment variables:
```bash
export COS_LLM_PROVIDER=deepseek
export COS_LLM_API_KEY=your_deepseek_api_key
export COS_LLM_MODEL=deepseek-v4-pro
```

## Backwards Compatibility

- All existing endpoints remain functional
- LLM features are optional and fallback to built-in logic
- UI maintains compatibility with previous versions

## Known Issues

- LLM calls may timeout; system gracefully falls back
- Requires DeepSeek API key for enhanced features</content>
