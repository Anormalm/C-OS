# C-OS v0.1.0

Release date: 2026-03-21

## Summary

v0.1.0 is the first public launch package for C-OS with:
- temporal memory + hybrid retrieval foundations
- non-technical user workflow
- coach + weekly summary features
- quality dashboard + evaluation harness
- open-source governance and CI

## Included Sprint Scope

1. Sprint 1: publishability baseline
2. Sprint 2: onboarding + feedback + weekly summary
3. Sprint 3: evaluation harness + quality dashboard
4. Sprint 4: launch packaging (landing README, demo assets, sample dataset, release docs)

## Key Endpoints

- `POST /ingest/text`
- `POST /query/retrieve`
- `POST /coach/advice`
- `POST /summary/weekly`
- `POST /evaluation/run`
- `GET /quality/dashboard`

## Launch Assets

- README launch page
- demo recording guide + voiceover script
- sample dataset + loader script

## Known Limitations

- Neo4j adapter includes advanced queries that may require APOC in some setups.
- Advice generation is heuristic and should be tuned with production feedback.
- UI screenshots are placeholders until final capture.
