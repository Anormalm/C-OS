# Changelog

## [0.1.0] - 2026-03-21

### Added
- Initial C-OS architecture with ingestion, extraction, graph/vector memory, temporal reasoning, and inference.
- FastAPI API for ingestion, retrieval, temporal query, diagnostics, and coaching endpoints.
- Non-technical web workspace with timeline and coach flows.
- Contradiction tracking with versioned fact handling.
- Basic automated tests for retrieval, temporal correctness, and advice generation.
