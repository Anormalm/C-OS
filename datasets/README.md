# Sample Dataset

`sample_memory_notes.jsonl` is a starter dataset for demo and evaluation.

Each row includes:
- `text`
- `source_type`
- `source_uri`
- `valid_from`
- `metadata`

Use it to:

1. Seed C-OS quickly for local demos.
2. Validate onboarding, coaching, and weekly summary flows.
3. Run repeatable quality checks before release.
